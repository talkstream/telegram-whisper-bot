# Telegram Whisper Bot

**AI-бот для транскрипции голосовых сообщений в Telegram**

[![Version](https://img.shields.io/badge/version-5.0.0-blue.svg)](https://github.com/talkstream/telegram-whisper-bot)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Telegram](https://img.shields.io/badge/Telegram-@editorialsrobot-blue?logo=telegram)](https://t.me/editorialsrobot)

Голосовые, аудио и видео → отформатированный текст. Qwen3-ASR + Qwen LLM.

**[Попробовать →](https://t.me/editorialsrobot)**

---

## Быстрый старт

1. Откройте [@editorialsrobot](https://t.me/editorialsrobot)
2. Отправьте голосовое или аудио/видео файл
3. Получите текст через несколько секунд

При первом запуске — 15 минут бесплатно.

---

## Возможности

- **Диаризация** — авто-определение спикеров в аудио ≥60с
  - 3 бэкенда: DashScope two-pass, AssemblyAI, Gemini
  - Метки спикеров с возможностью отключения (`/speakers`)
- **Транскрипция** — голосовые, аудио, видео
- **Форматирование** — пунктуация, абзацы, «ё» через LLM
- **Смарт-чанкинг** — семантическая разбивка длинных текстов для LLM (по спикерам / абзацам)
- **Прогресс с ETA** — стадии обработки с оценкой времени
- **Загрузка больших файлов** — до 500 МБ через Mini App (`/upload`)
- **Импорт по ссылке** — Яндекс.Диск, Google Drive, Dropbox
- **Авто-файл** — тексты >8000 символов автоматически как .txt
- **52 языка** — автоопределение
- **Оплата в Stars** — Telegram-нативная

## Технические характеристики

| Параметр | Значение |
|----------|----------|
| ASR | Qwen3-ASR-Flash (REST), диаризация: DashScope / AssemblyAI / Gemini |
| LLM | Qwen3.5-397B (fallback: Gemini 3 Flash) |
| Синхронная обработка | до 15 сек |
| Асинхронная | 15+ сек (MNS), таймаут 600с |
| Смарт-чанкинг | семантическая разбивка >4000 символов для LLM |
| Прогресс | ETA по стадиям (загрузка → ASR → диаризация → LLM → доставка) |
| Регион | EU (Франкфурт) |
| Cold start | 2-3 сек |

## Changelog

### [5.0.0] — 2026-02-20
- **Журналистский pipeline**: обработка интервью, пресс-конференций, лекций
- Смарт-чанкинг LLM: семантическая разбивка по спикерам / абзацам с overlap-контекстом
- ProgressManager с ETA по стадиям обработки
- Авто-файл: тексты >8000 символов → `.txt` с таймштампом
- Time budget watchdog: пропуск LLM при нехватке времени FC
- Качество диаризации: фильтр микросегментов, gap ratio, оконная нормализация
- Динамический таймаут диаризации (180-300с по длине аудио)
- `/upload` — Mini App для загрузки файлов до 500 МБ (OSS signed URL)
- Импорт по ссылке: Яндекс.Диск, Google Drive, Dropbox
- Пакет «Редакция» (3000 мин / 1399 ⭐)
- FC timeout 300→600с
- 474 теста

### [4.3.0] — 2026-02-17
- Параллельная обработка пересланных аудиосообщений (порог 60→15с)
- Быстрый путь в audio-processor: аудио <60с без диаризации
- Обнаружение реальной длительности документов с duration=0
- 230 тестов
