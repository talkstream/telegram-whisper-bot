14 — Exit-стратегия

Оценка ₽120-180M

VK — приоритетный покупатель

  • VK Tech IPO 2025 — активно скупают технологии
  • VK Видео/Клипы — нужны субтитры
  • VK Мессенджер — нужен STT для голосовых
  • Нет своего STT (в отличие от Яндекса и Сбера)
  • Наша Telegram-база → миграция в VK

Что строим для exit

  • REST API — интеграция в экосистему
  • Python/JS SDK для разработчиков
  • On-premise вариант (Docker)
  • Реестр российского ПО
  • Документированная архитектура
